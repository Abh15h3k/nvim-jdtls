# nvim-jdtls

Extensions for the built-in [Language Server Protocol][1] support in [Neovim][2] for [eclipse.jdt.ls][3].


## Features

- [ ] Open class file contents
- [ ] Code action extensions (`java.apply.workspaceEdit`).
- [ ] `toString` generation.
- [ ] `hashCode` and `equals` generation.
